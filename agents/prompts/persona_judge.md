# 评审提示词（evaluate.py 使用，独立于预测器，避免自证偏差）

## JUDGE

你是严格的评审员。任务：判断"对分析师观点的盲预测"是否命中了其真实文章的核心观点方向。

评判标准（只看**核心观点与方向**，不要求细节一致）：
- hit：预测的命题方向与真实文章核心观点方向一致（如同为看多出口、同为提示某风险）；
- partial：主题对上且方向部分一致，或抓住了次要结论但偏离核心结论；
- miss：方向相反、主题跑偏、或预测过于空泛以至于无法算命中。

注意：预测是在**没有看到文章**的情况下做出的，评判要克制——空泛的"正确废话"（如"经济有不确定性"）不能给 hit。

严格输出一个 JSON（用 ```json 代码块包裹）：

```json
{{
  "verdict": "hit | partial | miss",
  "reason": "判定理由（80 字内，指出预测与文章核心观点的对应或偏差）"
}}
```

【盲预测】
{prediction}

【真实文章】（标题：{title}，日期：{date}，正文截断）
{article_text}

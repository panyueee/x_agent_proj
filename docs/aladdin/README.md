## 每篇核心新发现（相对初版 survey）

### 01 历史与商业
- **Technology Services 收入 2007-2025 全部从 SEC 10-K 原文确认**（初版只有一个模糊数字）：$190M(2007) → $393M(2008, 危机年 +107%) → $974M(2019) → $1,603M(2024) → $1,981M(2025, 含 Preqin ~$210M)；并厘清三次口径改名与 2024 年控股重组（新 CIK 2012383）。
- 澄清初版的混淆："2025 增速 24%"是收入（含 Preqin 并表），有机 ACV 增速是 +16%；**ACV 绝对值 10-K 从未披露**，~$2B 是三方拼接。
- Maiden Lane I/II/III 纽约联储一手数字：纳税人净收益合计约 $119 亿，BlackRock 首年费用 $45.3M+$25.5M，无投标；2020 疫情合同 ~$40M/年，2022-05-02 终止。
- 监管三线补全：FSOC 2014-07-31 放弃 SIFI 指定、欧盟 Ombudsman 裁决原文（€280k 合同"报价异常之低"）、Economist monoculture 原文引语。

### 02 风险方法论
- **谱系钉死**：官方披露自研股票模型名为 **BFRE World**（暴露 z-score 标准化 + 横截面回归）= 标准 Barra 式；与固收 KRD/利差因子拼进同一协方差矩阵。
- **压力测试传导数学是官方署名披露而非行业推测**：Golub-Greenberg-Ratcliffe (JPM 2018) 的 Market-Driven Scenarios 框架——政策因子冲击经协方差条件期望传导 + Mahalanobis 距离/情景 Z-score 合理性检验 + slow(5y)/fast(1y)/recent 三档协方差窗口。
- 罕见一手信用模型解剖：BlackRock 为 NAIC 出的 Non-Agency RMBS Methodology (2015)——贷款级变量清单（Incentive/Burnout/Seasonality）、动态聚类、拖欠状态转移矩阵、损失严重度公式；另有 Fed FOMC 2010 备忘录直接采用其提前偿付模型的旁证。
- 官方方法论教科书：*BlackRock's Guide to Fixed-Income Risk Management*（Wiley 2023，Golub 主编）19 章目录 + 第 1 章 IRMP 五支柱全文（双边 RAG 风险区间等细节）。
- 明确列出无公开披露清单：EWMA 半衰期、蒙特卡洛路径数、完整因子清单、真实 Green Package 样例等。

### 03 技术架构
- **BMS 消息总线的专利级细节**（US10341196B2 热升级 / US10263855B2 消息层鉴权）+ 七语言官方绑定名单；"One BlackRock"规则族完整版。
- **QCon London 2026 演讲（最硬的计算规模一手口径）**：Aladdin Wealth 每晚 1,500 万+ 组合全量风险计算、3,000+ 因子稠密协方差、E·C·Eᵀ + "收盘冻结输入、缓存计算图中间量"的架构手法；Hive/Solr/Cassandra 按访问模式多投影存储。
- **Argo Events 诞生于 BlackRock 并捐给 CNCF**（2022 随 Argo 毕业）；Azure 迁移演练细节（一年内生产环境进出十余次）；2024-25 年 ML 交易路由专利族。
- **Julia 定位修正**：确证 2014 起用于 analytics 且有 BMS 绑定，但发现"Julia 原型 → Rust 生产"反例（lcso 库），近年 JD 零提及——判定存量维持、增量停滞（初版"金融业最著名 Julia 生产用户"需加时态限定）。
- "6,000 台计算机 / 30,000 组合"等流传数字全部溯源到 The Economist 2013 原文（2013 时点数据，云迁移后已失效）；"每天数十亿次计算"溯源不明，不建议引用。

### 04 竞品对照
- 最硬的定价挖掘：JP Morgan 接入费 $1.5M、年费涨至 $5.3M/年（[低/中]）；倒推平均 ACV ≈ $1.5-2M/客户/年；Bloomberg Terminal $31,980/年/席是全场唯一准公开价。**没有任何一份带金额的 Aladdin 政府采购合同可查**（如实说明）。
- 反向案例链：LACERA（$58B）公开以 "groupthink" 为由拒绝 Aladdin；智库主张拆分 Aladdin 为系统重要性市场设施。
- 被忽略的正面战场：Barclays POINT 关停（2016-2018）迫使数百家固收资管重新选型，Aladdin/PORT/Yield Book 三方截流。
- 功能矩阵结论——Aladdin 三个结构性短板：股票因子非行业通用口径（量化圈语言在 Barra/Axioma 手里，解释了 MSCI Analytics $675M 收入 + ~50% 利润率为何杀不死）、基金会计/多准则报表不如 SimCorp、实施重（12-24 个月）且贵。

### 05 个人版路线图
- **提案 1（风险日报）比初版预想容易得多**：项目里已有 `x_agent/factor_model.py`（toraniko，factor_returns 表 85 天因子收益）、`sw_sector_cache`（5609 只 GICS 分类）、`backtest/data.py` 干净的 parquet 加载层——只缺暴露回归 + EWMA 协方差 + 风险分解三块纯数学。
- **关键阻塞项**：库里没有任何持仓表 → 把提案 3 拆成 3a（securities/portfolios/positions 主表切片，前置）+ 3b（信号绩效，signals 仅 260 行，延后等数据）。
- 意外之喜：`bond_history`（中美利率曲线）与 `fx_history`（美元指数）parquet 让提案 4 的宏观维度基本免费。
- **修正后的顺序：1（含 3a）→ 2 → 3b → 4 → 5**，三个迭代批次约 5 / 4.5 / 4 天，每批附验收标准。

## 给本项目的 Top 3 优先级建议（详见 05）

1. **批次一：提案 3a + 提案 1**（约 5 天）——建 securities/portfolios/positions 主表 + 全历史因子库 + 组合风险分解日报（`output/risk_report.md` + digest 风险区块）。这是把"信号→组合→风险"闭环立起来的地基，且 factor_model.py 等半套基础已在。
2. **批次二：提案 2 + 3b**（约 4.5 天）——危机场景重放（2008/2015/2016/2018/2020/2022/2024 七情景，本地 2.1 万标的全历史是独有筹码）+ 信号绩效回填管道（先建管道等数据，不急于出结论）。
3. **工程模式即刻可借鉴**（不需等排期，见 03 §15）："收盘冻结 + 计算图缓存"（每晚预计算因子协方差与个股 beta，白天信号来了秒级算边际风险）、"主数据一份、派生投影多份"、"信号入库强制带 source/trace_id"。

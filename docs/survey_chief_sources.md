# 券商首席分析师历史文章/研报 —— 补充渠道调研

调研日期：2026-07-02
调研范围：在已有渠道（公众号 via WeWe RSS、东方财富研报库、知识星球、乌拉邦、格隆汇初探）之外，
系统排查还能从哪里成规模、成历史地拉到中国券商首席的文章/研报。

目标人物：张瑜(华创宏观)、李迅雷(中泰)、郭磊(广发)、戴康(广发)、明明(中信)、
牟一凌(民生)、罗志恒(粤开)、钟正生(平安)、熊园(国盛)、陈果(中信建投) 等。

> 说明：本文档只做可访问性/历史深度/免费边界/接口形态的实测调研，**不写任何入库代码**，
> 不碰 output/rag.db，不注册任何账号。所有测试用 curl/requests 单发、克制频率。

## 逐渠道详情

### 1. 股票报告网 nxny.com（★ 最高性价比，强烈推荐）

- **可访问性**：HTTP 200，纯静态 HTML，无 JS 渲染、无签名、无验证码。
- **券商覆盖**：首页导航即按券商分类（`/stype_{id}/`）。实测已确认收录：
  - 华创证券 `stype_61`、中泰证券 `stype_1`、中信建投 `stype_66`、国盛证券 `stype_558`、
    粤开证券 `stype_566`、平安证券 `stype_63`、天风证券 `stype_550`、华泰 `stype_19` 等 40+ 家。
  - **广发证券**：首页出现 10 次并带 `view_` 详情链接，确认收录（导航未单列，走站内检索/首页入口）。
  - 中信证券、民生证券：首页以文本形式出现，**收录情况待进一步确认**（导航未单列，站内 search.php 未命中，可能用了不同分类 id）。
- **历史深度**：华创页 `/stype_61_p{N}/` 规则分页，每页 70 条。翻到 p200 已见 2025-03 的报告，
  p600/p800 仍返回内容（日期字段解析待细化，但**至少可回溯一年多，疑似数年**）。
- **免费边界（关键优势）**：详情页 `/report/view_{id}.html` 直接 SSR 渲染出**研报摘要+正文可见文字约 2600 字**
  （实测张瑜《6月PMI点评》可读到完整正文段落）。仅 **原始 PDF 标注"此报告为加密报告"需登录下载**。
  也就是说 **不登录就能拿到标题、作者、日期、类型、评级、页数、以及正文文字**——对 RAG 入库已经足够。
- **接口形态**：列表 `https://www.nxny.com/stype_{id}_p{N}/`；详情 `https://www.nxny.com/report/view_{id}.html`。
  纯 HTML 正则即可解析，字段规整（`时间：` `类型：` `作者：` `来源：` `下载权限：`）。
- **反爬/风险**：极低。备案 `蜀ICP备15031742`，Copyright 2005-2021，老站稳定。合规上属第三方转载，注意频率与 robots。
- **接入难度**：★☆☆ 低。requests + 正则/bs4 即可，无需 Playwright。

**这是本次调研最重要的发现**：它同时满足"收录东财缺的四家（华创/中泰/广发已确认）+ 免费拿到正文文字 + 无反爬"。

### 2. 格隆汇 gelonghui.com（张瑜历史文章最优来源）

- **已验证**：用户 `293015` 确认就是 **张瑜（华创宏观）**——`/user/293015/article` 页 SSR 出的文章标题
  （《1月非农为何超预期？》《中国版QE：谁在"非常规"扩表？》等）和 grep 到的"华创/宏观"标签吻合。
- **历史深度**：`/user/293015/article?page=N` 分页，每页 15 篇，翻到 page20、page40 仍满页返回，
  **粗估 600+ 篇，跨越多年**。
- **免费边界**：单篇 `/p/{id}` **全文免费无墙**（实测无"登录后/付费/VIP/阅读全文"截断，渲染出 34 万字符含完整正文）。
- **接口形态**：Nuxt SSR。列表页 HTML 里 `"/p/{id}"` 即文章链接；未找到公开 JSON 列表 API
  （猜测的 `/api/v3/users/.../articles` 全 404），但 **直接抓 SSR 的 HTML 分页就够**，无需 API。
- **反爬/风险**：低。SSR 直出，无签名。
- **接入难度**：★★☆ 中。需要先定位每个首席的 user id（张瑜=293015 已知，其余要在站内搜索确认），再翻页抓 `/p/{id}`。

### 3. 发现报告 fxbaogao.com

- **可访问性**：HTTP 200。
- **券商覆盖**：有机构归档页 `/archives/organization/{券商名}`，实测华创证券页可列研报；
  搜索确认**华创/中泰/广发/中信均有机构页**（东财缺的四家全覆盖）。
- **历史深度**：机构页分页可翻到千页量级，历史深。
- **免费边界**：**列表+摘要免费，PDF 全文需 VIP/付费社群**（页面有 `/seo/vip`、`/seo/wxrss`、`购前协议`）。
  作者名在列表里被星号打码（"徐***"），不利于按人筛选。
- **接口形态**：机构页 HTML；有站内接口但需进一步抓包。
- **风险/难度**：★★☆ 中。免费只能拿列表+摘要，正文要花钱，价值弱于 nxny。

### 4. 慧博投研 hibor.com.cn

- **可访问性**：HTTP 200，全券商研报聚合（业内头部）。
- **免费边界（劣势）**：详情页 `/data/{hash}.html` 正文走 AJAX `/hiborweb/DocDetail/NewContent?ncid=...`，
  实测**该接口需鉴权，未登录返回 302 "Object moved"**；页面反复提示"登录访问""下载安装慧博PC客户端解锁完整功能"。
  即列表可见，**正文/PDF 强登录墙**。
- **接口形态**：HTML + 需鉴权 AJAX。
- **风险/难度**：★★★ 高。登录态 + 客户端门槛，性价比低于 nxny。

### 5. 新浪财经意见领袖 finance.sina.com.cn/zl

- **可访问性**：HTTP 200。多位首席有独立作者页，形如 `author.d.html?uid={uid}`。
  已知：**李迅雷 uid=1803814487（《迅雷观察》）**、张瑜 uid=1705952971（页面标题《热评中国》专栏）。
- **历史深度**：作者页 `?page=N` 分页；李迅雷文章可回溯到 2021、2022 及更早（搜索命中多篇 2021-2022 旧文），历史深。
- **免费边界**：**意见领袖专栏文章全文免费**（媒体属性，无付费墙）。
- **接口形态**：作者页 HTML + `page=` 翻页。正文文章为 `.../doc-xxx.shtml` 或 `zl-xxx.shtml`。
  注意：文章链接是 JS/接口异步加载，静态 HTML 里直接 grep `doc-` 命中为 0，**需定位 feed 接口或渲染**（页面有 `column/view/order_list.html` 线索，待验证）。
- **风险/难度**：★★☆ 中。作者 uid 需逐个确认；列表可能要抓 feed 接口。适合作为"意见领袖类随笔"补充源。

### 6. 中国首席经济学家论坛 chinacef.cn（技术已攻破，但需签名，谨慎）

- **可访问性**：HTTP 200，前端是 Vue SPA，数据走**加密 JSON API**。
- **已逆向出加密方案**（记录备查）：
  - `/api/ExpertApi/ExpertList`、`/api/ArticleApi/ArticleListV2`、`/api/ArticleApi/Detail`、`/api/ExpertApi/ExpertDetail` 等。
  - 请求/响应用 **DES-CBC/Pkcs7** 加密，Base64 后把 `/`→`@@`、`+`→`$$`。
  - 业务密钥（key=iv，8字节）：`HFaJjnRX`（配 token `5Mlp6YvmuuTjq8uY`）或 `cYg6v3yi`（配 `Q2dgHi76kxulu4gU`），
    存于 localStorage；请求参数含 `token = 时间戳+"&"+token`。
  - 实测用 `HFaJjnRX` 成功解出干净 JSON（返回 `{"s":401,"d":"token错误"}`，说明加解密链路正确，只差合法 token 组合）。
- **人物覆盖**：李迅雷是论坛副理事长，站上应有其专栏；具体文章数/年份**未验证**（受阻于 token 校验，未继续爆破，避免触发风控）。
- **风险/难度**：★★★ 高。需要正确复刻 token 生成逻辑，且签名机制随时可能变。**价值待评估**——李迅雷在新浪/中泰渠道已可覆盖，这里优先级低。

## 按性价比排序的接入建议

1. **首选：股票报告网 nxny.com** —— 唯一"免费拿到正文文字 + 覆盖东财缺的四家 + 零反爬 + 纯静态易解析"的渠道。
   建议做成主力研报源：按 `stype_{券商id}` 遍历分页 → 抓 `view_{id}` 详情页正文入库。先补齐中信/民生的分类 id 确认。
2. **张瑜专用：格隆汇** —— 张瑜历史文章（含多年长文、全文免费）的最优来源。按 `/user/{id}/article?page=N` 翻页 + `/p/{id}` 抓全文。
   需先为其他首席逐个确认 user id。
3. **意见领袖随笔：新浪财经 zl** —— 首席"观点随笔类"全文免费、历史深，适合补 nxny（研报体）之外的评论文章。需解决 feed 接口翻页。

> 补充：fxbaogao/hibor 作为"元数据/发现"用途可以，但正文都要钱或要登录，不适合做免费正文入库主力。
> chinacef 技术上已打通加解密，但 token 校验未过、且人物可被其他渠道替代，**列为低优先备选**。

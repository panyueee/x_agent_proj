## 二、评估维度说明

- **真实度**：有无实盘赛参赛记录/交割单/实盘帖，还是纯观点输出。
- **活跃度**：主页"全部日志"最近发帖日期 + 近 90 天主帖数（`n90d`，2026-04-03 起算；主帖口径，不含跟帖）。
- **信息价值**：复盘逻辑/产业梳理 > 大盘观点 > 情绪鸡汤 > 荐股喊单。
- **风格标签**：取自主页"创作风格"官方标签 + 帖子内容判断。
- **风险**：红包帖导流、直播/训练营收费带单、仿号。

## 五、注意事项

1. **红包帖水分**：米开朗基瑞、小土堆爆金币、作手阿意等的高回复量部分由"红包帖"机制刺激，
   回复数不能直接当热度信号；湖南人回复数低（十几条）但浏览 3-4 万，才是真实阅读型账号。
2. **抓取频率**：11 个用户按现有 `blog_since` 模式轮询即可；本次实测 3.5s 间隔无任何风控，
   保持现有节奏不要加速。
3. **名人堂/人气榜登录墙**：后续如需拿官方人气榜数据，需要登录态 cookie；不建议为此注册爬号，
   靠首页推荐位+实盘赛榜季度性人工复查候选池即可。
4. **仿号防坑**：加新用户前务必核对数字 id 而非昵称搜索（心跳r/小鳄鱼/92科比全是重灾区）。
